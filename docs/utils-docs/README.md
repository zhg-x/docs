# common-utils - v0.5.0

## 介绍

[`@zhg-x/common-utils`](https://www.npmjs.com/package/@zhg-x/common-utils)包主要提供一些经常用到的ts、js工具方法。

## 安装

通过npm：
```
npm i @zhg-x/common-utils
```



## 使用

可以直接通过方法名称调用对应方法，也可以通过类名`.`方法名称调用对应的方法。

```
add(); // 1.直接通过方法名称调用
NumberUtils.add(); // 2.通过类名.方法名称调用
```
